> Simple API for WBC MGR

API of WBC manager based on node.js, express, mongodb ...


## Installation

Install with [npm](https://www.npmjs.com/)

```sh
$ npm install
```


## Usage

Run with [npm](https://www.npmjs.com/)

```sh
$ npm run start
```
